# 描述

该项目基于 Flink 2.2.0 给出了 [DataStream API](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/datastream/overview/ "DataStream API") 和 [DataStream API (V2)](https://nightlies.apache.org/flink/flink-docs-release-2.2/docs/dev/datastream-v2/overview/ "DataStream API (V2)") 两种 API 的示例，如下，

- DataStream API 批处理
    - file4batchv1
- DataStream API 流处理
    - file4streamv1
- DataStream API (V2) 批处理
    - file4batchv2
- DataStream API (V2) 流处理
    - file4streamv2
